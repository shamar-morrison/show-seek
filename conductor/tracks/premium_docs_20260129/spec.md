# Specification - Create Premium Features Reference Document

## Context
The user is building a Next.js web version of the ShowSeek mobile app. To ensure parity in premium offerings, a clear reference document is needed.

## Goal
Generate a `premium-upgrades.md` file that lists all categories and features currently defined in the mobile app's source code (`src/constants/premiumFeatures.ts`).

## Requirements
- Maintain category grouping.
- Include Feature Title.
- Include Feature Description.
- Include Icon names (for reference).
- Mark "New" features.
